# Electronics Network Management System

## Описание проекта
Веб-приложение для управления сетью по продаже электроники с иерархической структурой (заводы, розничные сети, ИП).

## Технологии
- Python 3.8+
- Django 3.2+
- Django REST Framework 3.14+
- PostgreSQL 10+
- DRF-YASG (Swagger документация)
